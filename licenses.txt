All code is licensed under the GNU GPL Version 2 open source software license. Code is defined as anything contained within
the src/ and lua/ directories. Code includes both C++ and Lua source files.

All original Allacrost content is licensed under the GNU GPL Version 2. Content is defined as anything contained within the img/,
snd/, and mus/ directories. Content includes PNG and JPEG images, WAV files, and Vorbis OGG music files.
